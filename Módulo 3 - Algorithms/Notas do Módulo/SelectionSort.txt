SELECTION SORT

    COMO SE PODE ESPERAR, É UM ALGORITMO QUE ORDENA UM CONJUNTO DE ELEMENTOS. E O AGORITHM RECALL É UM CONJUNTO DE
    PASSO A PASSO PARA CONCLUIR UMA TAREFA. EM SELECTION SORT A IDEIA BÁSICA É A SEGUINTE: ENCONTRAR O MENOR ELEMENTO
    NÃO ORDENADO E ADICIONÁ-LO AO FINAL DA LISTA ORDENADA.

    EM PSEUDOCÓDIGO TEMOS:

        REPITA ATÉ QUE NÃO HAJA ELEMENOS NÃO ORDENADOS:
            PESQUISE NOS DADOS NÃO ORDENADOS PARA ENCONTRAR O MENOR VALOR
            TROQUE O MENOR VALOR PELO PRIMEIRO ELEMENTO DA PARTE NÃO ORDENADA

    NO PIOR CENÁRIO: TEMOS QUE EXAMINAR TODOS OS ELEMENTOS DO ARRAY PARA ENCONTRAR O MENOR ELEMENTO NÃO ORDENADO
    E TEMOS QUE REPETIR ESSE PROCESSO N VEZES. UMA VEZ PARA CADA ELEMENTO DO ARRAY, PORQUE NESSE ALGORITMO, NÓS SÓ
    ORDENAMOS UM ELEMENTO DE CADA VEZ.

    NO MELHRO CENÁRIO: NA VERDADE, QUASE A MESMA COISA, POIS AINDA TEMOS QUE EXAMINAR TODOS OS ELEMENTOS DO ARRAY,
    PARA CONFIRMAR QUE É, DE FATO, O MENOR ELEMENTO.

    NO PIOR DOS CASOS TEMOS: O(n²)
    NO MELHOR DOS CASOS TEMOS: Ω(n²)
